📱 Pixel Shop — Teste Aparelhos na Nuvem
Plataforma moderna para testar dispositivos virtualizados

A Pixel Shop é um site desenvolvido para permitir que usuários testem aparelhos virtualmente na nuvem, explorando funções, aplicativos e desempenho em tempo real.
Este projeto inclui a página inicial, a página de cadastro, a página de confirmação e scripts de validação.

🚀 Funcionalidades Principais

✔️ Teste de aparelhos virtualizados
✔️ Interface moderna e responsiva
✔️ Página inicial com catálogo
✔️ Formulário de cadastro com validação em JavaScript
✔️ Página de confirmação com animação
✔️ Layouts com gradientes e identidade visual sólida

📝 Detalhes Técnicos
🔹 Validação do Formulário

O arquivo script.cadastro.js implementa validações para:

Nome

Email

Senha

CPF

Gênero

Cada campo possui estados: erro (vermelho) e sucesso (verde), exibindo feedback ao usuário.
